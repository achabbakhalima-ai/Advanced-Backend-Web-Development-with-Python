# 🐍 Travaux Pratiques Python

## 📖 Description
Ce dépôt GitHub contient l’ensemble de mes **travaux pratiques (TP) réalisés en Python** dans le cadre de ma formation en développement informatique.

Ces exercices m’ont permis de mettre en pratique plusieurs concepts importants tels que :

- 🧠 les bases du langage Python
- 🧩 la programmation orientée objet (POO)
- 🏗️ l'organisation d’un projet Python
- 🌐 l'introduction au framework web **Django**

Chaque dossier correspond à un TP spécifique avec son code source et ses fichiers nécessaires à l'exécution.

---
## 🧪 Contenu des travaux pratiques

### 🔹 TP1 : tp1_python
Ce TP contient des exercices permettant de pratiquer les **fondamentaux du langage Python** :

- variables
- structures conditionnelles (`if`)
- boucles (`for`, `while`)
- fonctions
- manipulation de listes et dictionnaires

---

### ☕ TP2 :tp2_systeme_boisson
Ce TP consiste à développer un **petit système représentant les boissons dans un café**.

Fonctionnalités principales :

- création de boissons avec **description et prix**
- ajout d’ingrédients (**lait, sucre, caramel**)
- utilisation du **Design Pattern Décorateur**
- combinaison de boissons
- gestion des clients
- gestion des commandes
- système de **points de fidélité**

---

### 🌐 TP3 : tp3_django
Ce TP permet de découvrir les bases du framework web **Django** :

- création d’un projet Django
- lancement du serveur de développement
- création d'applications avec `startapp`
- gestion des migrations
- structure d'un projet web Django
