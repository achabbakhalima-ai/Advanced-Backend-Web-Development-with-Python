# Travaux Pratiques Python

## Description
Ce dépôt contient l’ensemble de mes **travaux pratiques (TP) réalisés en Python** dans le cadre de ma formation en développement informatique.  
Ces exercices m’ont permis de pratiquer plusieurs concepts importants du langage Python et du développement logiciel.

Les principaux objectifs de ces TP sont :
- comprendre la programmation en Python
- appliquer la programmation orientée objet (POO)
- apprendre à organiser un projet Python
- découvrir le framework web Django

---

## Contenu du dépôt

Ce dépôt est organisé en plusieurs dossiers correspondant aux différents TP.

### TP1 : tp1_python
Ce TP contient des exercices permettant de pratiquer les bases du langage Python :
- variables
- structures de contrôle (if, for, while)
- fonctions
- manipulation de listes et dictionnaires

### TP2 :tp2_systeme_boisson
Ce TP consiste à développer un petit système représentant les boissons dans un café.

Fonctionnalités principales :
- création de boissons avec description et prix
- ajout d’ingrédients (lait, sucre, caramel)
- utilisation du **design pattern Décorateur**
- combinaison de boissons
- gestion des clients
- gestion des commandes
- système de points de fidélité

### TP3 : tp3_django
Ce TP permet de découvrir les bases du framework **Django** :

- création d’un projet Django
- lancement du serveur de développement
- création d’applications avec `startapp`
- utilisation des migrations
- structure d’un projet web Django
